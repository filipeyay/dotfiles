# Dotfiles Public

Here you'll find my really simple dotfiles.

- Emacs config uses [DOOM Emacs](https://github.com/doomemacs/doomemacs)
- Neovim config uses [Lazyvim](https://www.lazyvim.org/)
- Tmux config uses [tpm](https://github.com/tmux-plugins/tpm)
- Bash config uses [Oh My Bash](https://github.com/ohmybash/oh-my-bash)

I use [BlexMono Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/IBMPlexMono.zip) as my default monospace font.
